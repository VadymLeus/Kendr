# 🚀 Kendr: NoCode платформа конструювання вебресурсів

**Kendr** — це веб-платформа, що дозволяє користувачам легко створювати та налаштовувати власні міні-сайти. Платформа надає готові шаблони, які можна наповнювати своїм контентом без навичок програмування.

## ✨ Основні Можливості

- **Реєстрація та Автентифікація**: Користувачі можуть створювати акаунти, оновлювати профілі та завантажувати аватари.
- **Створення Сайтів**: Простий та інтуїтивний процес створення сайтів на основі готових шаблонів.
- **Кастомізація Контенту**: Редагування заголовків, текстів, додавання товарів, категорій та логотипів.
- **Шаблон "Магазин"**: Повноцінна можливість створювати товари, ділити їх за категоріями та додавати до кошика.
- **Адмін-панель**: Спеціальний інтерфейс для адміністраторів для керування всіма сайтами на платформі.
- **Публічний Каталог**: Усі опубліковані сайти доступні для перегляду в загальному каталозі з можливістю пошуку.

---

## 🛠️ Технологічний Стек

### Backend
- **Node.js** та **Express**: Серверна логіка та API.
- **MySQL**: Реляційна база даних для зберігання даних користувачів, сайтів, товарів.
- **JWT (JSON Web Tokens)**: Для безпечної автентифікації користувачів.
- **Multer & Sharp**: Для завантаження та обробки зображень (аватари, логотипи, фото товарів).

### Frontend
- **React**: Основна бібліотека для побудови користувацького інтерфейсу.
- **React Router**: Для навігації між сторінками.
- **Axios**: Для взаємодії з backend API.
- **React Context**: Для глобального керування станом (автентифікація, кошик).

## ⚙️ Встановлення та Запуск

### Вимоги
- [Node.js](https://nodejs.org/)
- [MySQL](https://www.mysql.com/)

### 1. Налаштування Backend

1.  **Клонуйте репозиторій:**
    ```bash
    git clone https://github.com/VadymLeus/Kendr
    cd your-project-folder
    ```

2.  **Перейдіть у папку `backend` та встановіть залежності:**
    ```bash
    cd backend
    npm install
    ```

3.  **Налаштуйте базу даних:**
    - Запустіть ваш MySQL сервер.
    - Виконайте SQL-скрипт `database_schema.sql`, щоб створити базу даних `KendrDB` та всі необхідні таблиці.

4.  **Створіть файл `.env`** у папці `backend` та заповніть його даними для підключення до БД:
    ```env
    DB_HOST=localhost
    DB_USER=root
    DB_PASSWORD=your_db_password
    DB_NAME=KendrDB
    JWT_SECRET=your_super_secret_jwt_key
    PORT=5000
    ```

5.  **Запустіть сервер:**
    ```bash
    npm run dev
    ```
    Сервер буде доступний за адресою `http://localhost:5000`.

### 2. Налаштування Frontend

1.  **Відкрийте новий термінал, перейдіть у корінь проєкту та встановіть залежності для фронтенду:**
    ```bash
    cd ../frontend
    npm install
    ```

2.  **Запустіть клієнтську частину:**
    ```bash
    npm run dev
    ```
    Сайт відкриється у вашому браузері за адресою `http://localhost:5173`.
