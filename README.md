# Kendr: NoCode платформа конструювання вебресурсів

Kendr – це сучасна веборієнтована NoCode платформа, яка дозволяє користувачам без навичок програмування створювати повноцінні адаптивні вебресурси та інтернет-магазини. Система базується на компонентному підході та концепції **Schema-Driven UI**, що забезпечує збереження структури сторінок у форматі JSON. Такий підхід гарантує гнучкість контенту, генерацію семантично коректного коду та відсутність жорсткої прив'язки до інфраструктури постачальника (*Vendor Lock-in*).

Проєкт розгорнуто на платформі Vercel. Ознайомитися з роботою системи можна за посиланням: [https://kendr.online/](https://kendr.online/)

### Візуальний редактор

* Інтуїтивний Drag-and-Drop редактор сторінок.
* Миттєве відображення змін у режимі реального часу.
* Набір готових функціональних блоків для швидкого створення контенту.

* Захист приватних маршрутів та API.

### Система доступу

### Медіатека

* Інтеграція з Cloudinary.
* Завантаження та зберігання зображень.
* Автоматична оптимізація медіафайлів.

* Механізм апеляцій щодо заблокованих ресурсів.

## Налаштування Backend

### Перехід до директорії сервера

```bash
cd backend
```

### Встановлення залежностей

```bash
npm install
```

### Налаштування бази даних

1. Запустіть локальний сервер MySQL.
2. Створіть базу даних:

```sql
CREATE DATABASE KendrDB;
```

3. Виконайте SQL-скрипт:

```text
KendrDB.sql
```

для створення структури таблиць.

### Налаштування змінних середовища

Створіть файл `.env` у директорії `backend`.

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_db_password
DB_NAME=KendrDB

JWT_SECRET=your_super_secret_jwt_key

PORT=5000
```

### Запуск сервера

```bash
npm run dev
```

Після запуску сервер буде доступний за адресою:

```text
http://localhost:5000
```
